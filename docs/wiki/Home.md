## At a glance

- **Stack:** React 18, TypeScript 5.2, Vite 5, Tailwind CSS 3
- **Translation:** Google Translate free endpoint, client-side fetch
- **Deploy:** GitHub Actions → GitHub Pages at `/Universal-Subtitle-Translator/`
- **Privacy:** Files never leave your browser
